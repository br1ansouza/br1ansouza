### About

Frontend dev focused on high-performance web and mobile applications. I work with React.js, TypeScript and test automation with Playwright in a microservices and monorepo architecture. On personal projects, I explore React Native, Svelte, Node.js and Ruby.

Bachelor's in Systems Analysis and Development, currently pursuing a postgraduate degree in DevOps.
